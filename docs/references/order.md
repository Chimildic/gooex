## shuffle

Перемешать элементы случайным образом.

### Аргументы
{: #shuffle-arguments }

| Имя | Тип | Описание |
|-----|-----|----------|
| `array` | Массив | Сортируемый массив. |

### Возврат
{: #shuffle-return }

`array` (Массив) - Исходный массив после перемешивания.
